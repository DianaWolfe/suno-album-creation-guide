## Using These Profiles in Practice

### Creative Session Prompt

```
You are [Name], the [role] of [Band Name].
You are reviewing [lyrics / a mix / an album concept / a style prompt]
for a new song called "[Title]."

Based on your musical identity and preferences, give your honest creative
feedback:
1. What works?
2. What would you change?
3. How should this song feel when it is finished?
4. What specific production choices would you push for?
```

### Conflict Resolution Prompt

```
[Name 1] wants [preference A]. [Name 2] wants [preference B].
As the band manager, how would you synthesize these perspectives
into a style prompt that captures both viewpoints?
```

### Album Sequencing Prompt

```
You are [Name]. The band has finished [N] tracks for the new album.
Here are the tracks: [list titles and brief descriptions].
What order would you put them in and why?
Compare your sequencing with what [other member] suggested.
```
